# 🧮 Calculator Web App

A simple **web-based calculator** built using **HTML, CSS, and JavaScript**.  
It performs both **basic** and **advanced mathematical operations**.

---

## 🚀 Features
### 🔢 Basic Operations
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  

### 🧮 Advanced Operations
- √ Square Root  
- x² Square  
- ^ Power (exponent)  
- % Modulus (remainder)
